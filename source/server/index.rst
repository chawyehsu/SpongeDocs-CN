=================
创建服务器
=================

本文介绍如何建立与运行一个由 Sponge API 的实例驱动的服务器。

本节内容适合于那些希望创建 Sponge 服务器或是将已有服务器迁移到 Sponge
的用户。插件开发者或许也会对其中的部分章节内容感兴趣——服务器测试对插件开发是很有帮助的。

.. note::

    我们力图屏蔽所有实现细节，这份指南不针对任何一种具体的 SpongeAPI 实现。尽管如此，错误难免。
